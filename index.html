Prérequis

Pour ce tutoriel, nous partons du principe que vous connaissez le HTML, le CSS et bien-sûr le Javascript.

Si vous avez déjà jeté un coup d'oeil à Angular2, vous avez pu remarquer l'utilisation exclusif de TypeScript et non pas de Javascript pur. C'est pourquoi, les morceaux de code suivants sont redigés en TypeScript. Si vous ne connaissez pas ce sur-ensemble, faites un tour sur le site officiel.

Avant de débuter le tutoriel, installez node.js ainsi que npm sur votre ordinateur.
Installation

Commençons par installer ReactJs.
Ouvrez une invite de commande et tapez-y la commande suivante :

npm install -g @angular/cli Cette commande permet d'installer la CLI (Command-line Interface) d'Angular. Grâce à elle, nous pourrons créer facilement une application Angular, des squelettes de fichiers...
Création d'une application

Retournons dans notre invite de commande et tapons la commande suivante :

ng new nomAppli Remplacez nomAppli pour votre propre nom d'appli.
Placez vous dans le dossier de votre appli : cd nomAppli Vérifions que la création s'est bien déroulée et lançons le serveur d'Angular : ng serve et rendez-vous dans votre navigateur à l'adresse localhost:4200

Premier Composant

Entrons dans le vif du sujet: les composants ! Ils sont la base d'Angular2, les briques de votre application.

Angular vous en fournit un premier, nommé AppComponent, que vous trouverez dans le fichier app.components.ts. Tous les composants possèdent l'extension component.ts

Vous pouvez les créer à la main ou faire appel aux lignes de commande :

ng generate component nomComposant

Ouvrez le composant créé par Angular. Il possède déjà du code, un peu obscur au départ.

La première ligne est là pour importer un des nombreux modules d'Angular. Ici le module Component, toujours présent.

import { Component } from '@angular/core';

Vient ensuite un premier décorateur : @Component. Il sert de configuration à votre composant.

@Component({
selector: 'app-root', //défini le nom de la balise html qui appelra votre composant.
templateUrl: './app.component.html', //défini le fichier html nécessaire au composant.
styleUrls: ['./app.component.css'] //défini le fichier de style utilisé par votre composant.
})

Data Binding de base

Tout comme React, Angular peut effectuer des liens entre son modèle et ses vues. Cela s'appelle des binding

Commençons avec la forme le plus simple : afficher une variable du modèle dans la vue.

Two Way Data Binding

Il a fait la magie et le succès d'Angular 1 : le two-way data binding ! Cela signifie que si vous modifiez le modèle, la vue sera affectée, et que si vous modifiez la vue, le modèle sera affecté.

Il s'effectue grâce à l'attribut [(ngModel)] qui rassemble un binding de propriété (votre variable) et un binding d'évenement (ici, le changement de valeur de l'input)

Première directive : ngFor

Angular2 vous permet d'afficher des tableaux et autres liste de façon très simple, en utilisant une directive. Pour boucler sur une liste ou un tableau, il faut utiliser *ngFor.

La petite étoile correspond à un raccourci syntaxique, pour ne pas avoir à taper la vraie directive, beaucoup plus longue.

Les props
Les props sont comme les arguments d'une fonction, ils permettent de "paramêtrer" le composant.
Dans l'exemple suivant, on crée un bouton dont le texte est défini par la props "text".
On se retrouve donc avec un bouton contenant le texte "Nouveau texte". 

class Button extends React.Component {
  render() {
    return (
      <div className="myButton">
        {this.props.text}
      </div>
    );
  }
}

ReactDOM.render(<Button text="Nouveau texte" />, document.getElementById('buttonReactProps'));
La State
La state est l'état interne d'un élément.
Si le composant était une voiture cela serait le niveau de carburant, la vitesse courante, etc...
Ici la state contient une variable permettant de savoir si l'utilisateur a cliqué sur le bouton.
On ajuste alors la représentation graphique du bouton en fonction de cette dernière. 

class Button extends React.Component {
  state = {
    clicked: false
  }

  constructor() {
    super();
    this.handleClick = this.handleClick.bind(this);
  }

  handleClick() {
    this.setState({ clicked: !this.state.clicked });
  }

  render() {
    return (
      <div
        className="myButton"
        style={{ backgroundColor: this.state.clicked ? 'red' : 'blue' }}
        onClick={this.handleClick}
      >
        {this.props.text}
      </div>
    );
  }
}

ReactDOM.render(<Button text="Changement" />, document.getElementById('buttonReactState'));
